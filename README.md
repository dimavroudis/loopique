
# Loopique WP Theme

Theme for Beat's WordPress Developer Assignment

## Building tools

* Starter Theme: _s 
* Required Plugins: ACF Pro 
* Task Runner: npm scripts 
* Bundler: Webpack 
* Libraries Included: normalize.css, intersection-observer (pollyfill)

## Minimum Requirments

* WordPress 5.5
* PHP 7.0
